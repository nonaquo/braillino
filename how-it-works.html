How It Works

Transform your text into tactile Braille in four simple steps

1

Upload Arduino Code

First, you'll need to upload the Braillino code to your Arduino board:

  • Open Arduino IDE on your computer
  • Copy and paste the provided .ino code
  • Select your Arduino board type (Tools → Board)
  • Select the correct port (Tools → Port)
  • Click the upload button (→ arrow icon)
  • Wait for "Done uploading" message
2

Connect Your Device

Once your Arduino is programmed, it's time to connect it to the web app:

  • Keep your Arduino plugged into your computer via USB
  • Open the Braillino website in Chrome or Edge browser
  • Click the "Connect Device" button
  • Select your Arduino from the popup window
  • Wait for the status indicator to show "CONNECTED"
3

Type Your Message

Now you're ready to convert text into Braille:

  • Type any word or sentence (A-Z letters and spaces)
  • Maximum 100 characters per message
  • The text is automatically converted to uppercase
  • Press Enter or click "Send to Braille"
4

Experience Braille

Watch your Arduino bring the text to life:

  • The servos will move to form each letter in Braille
  • Each character displays for 4 seconds
  • Double letters (like "LL" in HELLO) show a brief gap
  • After the word finishes, enter a new message to continue

What You'll Need

Hardware

Arduino board (Uno or Nano) with 2 servo motors connected to pins 9 and 10

Software

Arduino IDE installed on your computer for uploading code to the board

Browser

Chrome or Edge browser (required for Web Serial API support)
